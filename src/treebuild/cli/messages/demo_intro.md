# Welcome to treebuild!

treebuild is a CLI tool for **sketching repository structures** before you build them.
Design your tree, then copy-paste it into a README, a prompt, or your notes.
When you are happy with the result, **materialize** it directly to your filesystem.
---

This demo will demonstrate the most basic commands.
1. Creating a new tree 
2. Setting the name of the root directory 
3. Adding / removing paths to / from  the tree 
4. Rendering it to text
5. Materializing it to the file system

---

At the end of this demo, you will be given the option to directly enter the `quickstart` mode.
