Start by planting your tree: 

```bash
treebuild plant
```

Then add/remove paths by growing/pruning the tree: 

```bash
treebuild grow <path1> <path2> ... 
```
OR 
```bash 
treebuild prune <path1> <path2> ...
```

When ready... render your tree by harvesting the fruits of your labour: 

```bash
treebuild harvest
```
