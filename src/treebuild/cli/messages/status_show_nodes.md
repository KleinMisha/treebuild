To remove a specific leaf / branch 

```bash 
treebuild prune <path1> <path2> ... 
```
or to prune the entire tree
```bash 
treebuild prune --all 

# or 

treebuild prune -a
``` 

Ready to harvest? 
```bash
treebuild harvest --renderer <RENDERER>

# or 

treebuild -r <RENDERER>
```
